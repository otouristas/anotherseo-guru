## 📏 **SPACING SYSTEM**

### **Padding Scale:**
```css
p-4    → sm:p-6    → lg:p-8    → xl:p-12
```

### **Margin Scale:**
```css
mb-4   → sm:mb-6   → lg:mb-8   → xl:mb-12
```

### **Typography Scale:**
```css
text-xs    → sm:text-sm    → lg:text-base
text-xl    → sm:text-2xl   → lg:text-3xl
text-3xl   → sm:text-4xl   → lg:text-5xl
```

### **Height Scale:**
```css
h-10   → sm:h-12   → lg:h-14   → xl:h-16
```
